This page provides a report of the assessment of the risk done on the new control system for the subsea plant.

Summary of the activities done

  • A checker was created to assess the design of the model after parsing the model from the given xml file.

  • Monte-Carlo Simulation was then performed to find the different statistics of the total duration.

  • In addition machine learning techniques like classification and regression are also applied to raise alerts.

  • A Inference of the performance of these techniques are also assessed.
